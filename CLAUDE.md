# open-graph-next

Using LLMs to improve the open, human-readable knowledge graph.

## Structure

- `accords/` - Reusable orientation documents ("accords") giving humans and LLM agents shared vocabulary for specific domains. Packaged as Claude Code agent skills. Based on the [agent-accords](https://github.com/daniloc/agent-accords) project.
- `all-the-opens/` - Rabbit Hole Browser: D3.js prototype exploring cooperative knowledge infrastructure across open sources
- `wikidata-SIFT/` - LLM-assisted Wikidata methodology testing and enrichment (merged from wikidata-llm-experiment)

Each subdirectory has its own CLAUDE.md with project-specific details.
